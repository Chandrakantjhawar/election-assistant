🤖 Groq API Key (AI Chat):

🏛️ Google Civic API Key (Find Reps):
1. Go to Google Cloud Console and create a New Project.
2. Search for "Google Civic Information API" and enable it.
3. Go to APIs & Services > Credentials, click + Create Credentials > API Key.
4. Copy the key (starts with AIza) and paste it above.
🔒 Privacy: Your keys are saved securely in your browser's local storage and never sent to a backend server.
